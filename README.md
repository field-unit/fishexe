# FISH.EXE — GitHub Pages package

This package contains the latest Temple Lake fishing game.

## Files

- `index.html` — the complete game
- `.nojekyll` — tells GitHub Pages to serve the files directly

The game currently streams the Field Unit demos from:

`https://field-unit.github.io/`

You do not need to upload separate MP3 files to this repository while those existing
audio URLs remain available.

## Publish on GitHub Pages

1. Create a new public GitHub repository, for example `fish-exe`.
2. Extract this ZIP.
3. In the repository, choose **Add file → Upload files**.
4. Upload `index.html` and `.nojekyll`.
5. Commit the files to the `main` branch.
6. Open **Settings → Pages**.
7. Choose **Deploy from a branch**.
8. Select:
   - Branch: `main`
   - Folder: `/ (root)`
9. Save the settings.

With a repository called `fish-exe`, the expected address is:

`https://field-unit.github.io/fish-exe/`

A new Pages deployment can take several minutes.
